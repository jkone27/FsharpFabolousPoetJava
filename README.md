# F# to Java via ASTs

the aim of this project is to try something similar as POC

```
.fs file ---> Fabolous AST ---> MAP ---> IVKM javaPoet ---> JAVA
```

* [JavaPoet](https://www.baeldung.com/java-poet)
* [Ikvm.net](https://github.com/ikvmnet/ikvm)
